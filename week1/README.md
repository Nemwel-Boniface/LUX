## Simpla array sum

> Given an array of integers, find the sum of its elements.

Constraints

Output Format

Print the sum of the array's elements as a single integer.

Sample Input

`6 `
`1 2 3 4 10 11`

Sample Output

`31`

Explanation

We print the sum of the array's elements:
1+2+3+4+10+11 = 21
